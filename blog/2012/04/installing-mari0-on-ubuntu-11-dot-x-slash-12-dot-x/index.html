Installing Mari0 on Ubuntu 11.x / 12.x

Posted on Apr 15th, 2012 under Games, Linux, Ubuntu

By Mari0, I mean this slice-of-genius Mario / Portal mashup.

sudo add-apt-repository ppa:bartbes/love-unstable sudo apt-get update sudo apt-get install love-unstable mkdir mari0 && cd mari0 wget http://stabyourself.net/dl.php?file=mari0-1006/mari0-linux.zip && unzip mari0-linux.zip love-unstable mari0_1.6.love
